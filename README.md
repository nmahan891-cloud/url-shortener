## فارسی

### 📌 معرفی پروژه

**کوتاه‌کننده لینک ماهان** یک پروژه کامل و حرفه‌ای برای کوتاه‌سازی لینک‌ها با قابلیت‌های پیشرفته است. این پروژه توسط یک برنامه‌نویس ۱۴ ساله ساخته شده تا به کسب‌وکارها کمک کند حضور آنلاین بهتری داشته باشند.

### 🚀 امکانات

- ✅ **کوتاه‌سازی لینک** با کد دلخواه یا تصادفی
- ✅ **سیستم ثبت‌نام و ورود** کاربران
- ✅ **داشبورد مدیریت** لینک‌ها
- ✅ **آمار کامل** کلیک‌ها با نمودار
- ✅ **QR Code** برای هر لینک
- ✅ **پشتیبانی از دو زبان** فارسی و انگلیسی
- ✅ **طراحی ریسپانسیو** برای همه دستگاه‌ها
- ✅ **قوانین و مقررات** و **حریم خصوصی**

### 🛠️ تکنولوژی‌ها

- **Flask** - فریم‌ورک وب پایتون
- **SQLAlchemy** - ORM برای دیتابیس
- **Flask-Login** - مدیریت احراز هویت
- **Flask-WTF** - فرم‌ها و اعتبارسنجی
- **QRCode** - تولید کدهای QR
- **Jinja2** - موتور قالب‌سازی

### 📁 نصب و راه‌اندازی

```bash
# 1. کلون کردن پروژه
git clone https://github.com/your-username/url-shortener.git
cd url-shortener

# 2. ایجاد محیط مجازی
python -m venv venv

# 3. فعال‌سازی محیط مجازی
# ویندوز:
venv\Scripts\activate
# لینوکس/مک:
source venv/bin/activate

# 4. نصب کتابخانه‌ها
pip install -r requirements.txt

# 5. ایجاد فایل .env
echo "SECRET_KEY=your-secret-key" > .env

# 6. اجرای پروژه
python run.py
